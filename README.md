# Telecom Churn Prediction Model

Machine learning model that predicts customer churn using ensemble learning methods.

## Overview
Built a predictive model to identify customers likely to leave a telecom service, achieving 76% recall on test data.

## Technologies
- Python
- scikit-learn
- pandas
- matplotlib

## Model Details
- Algorithms: Random Forest and Gradient Boosting
- Performance: 76% recall on test set
- Features: Customer demographics, service usage patterns, contract details

## Results
The model successfully identifies at-risk customers, enabling proactive retention strategies.
